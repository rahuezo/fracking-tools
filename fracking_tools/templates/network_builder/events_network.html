0. Before You Begin

Before building networks from events you MUST follow the exact instructions provided below.

Instructions

1. Your event files must be in .csv format.

2. Each event file must be a spreadsheet with the following format

Event 1 Event 2 Event 3
Person A Person B Person C
Person B Person C Person B
Person D Person D

3. Once you choose which files to upload and enter an output .zip name, you can click on Build >> to build your event networks.

4. A resulting output .zip file will be automatically downloaded to your computer.

5. Each event network will look something like the following

A B C D
A 0 1 0 1
B 1 0 1 1
C 0 1 0 1
D 1 1 1 0
