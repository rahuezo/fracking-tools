0. Before You Begin

Before building networks from individual pairs you MUST follow the exact instructions provided below.

Instructions

1. Your individual pairs files must be in .csv format.

2. Each individual pairs file must be a spreadsheet with the following format

Person 1 Person 2
Person A Person B
Person B Person C
Person A Person D

3. Once you choose which files to upload and enter an output .zip name, you can click on Build >> to build your event networks.

4. A resulting output .zip file will be automatically downloaded to your computer.

5. Each individual pairs network will look something like the following

A B C D
A 0 1 0 1
B 1 0 1 0
C 0 1 0 0
D 1 0 0 0
