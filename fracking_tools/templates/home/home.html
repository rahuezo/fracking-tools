Network Utilities

Build Networks from Events

An event network refers to a network where individuals are linked by attending the same events.

To build a network from events, you must provide a .csv spreadsheet where each column is an event and each row is an individual.

Build

Build Networks from Individual Pairs

To build a network from individual pairs, you must provide a .csv spreadsheet with only two columns: Individual A column and Individual B column.

In this spreadsheet, each row corresponds to a unique pair of individuals.

Compare Networks

You can compare network files (adjacency matrices) to determine the actor overlap, relationship overlap, unique actors, and the unique relationships.

Usually, the comparison involves a network built using meeting minutes and the other network using newspapers.

Document Utilities

Compare Documents

This utility allows you to compare two documents for content similarity and for textual similarity.

Content similarity uses the Vector Space Model (VSM) and Cosine Similarity to determine how similar two or a set of documents are to each other based on their topics.

Textual similarity uses Near Duplicate Detection to determine if two or a set of documents are almost identical. A numerical representation of textual similarity is given by computing the Jaccard similarity of documents.

Compare

Tag Documents for Keywords

This utility takes in one or a set of documents (.pdf, .docx, .txt) and a list of keywords. It then reads the documents and looks for the keywords within the document's text.

If any keywords are found, a .docx version of the document is created with a list of found keywords as the header of this document. Additionally, the found keywords are capitalized and boldened for easier human lookup.

Tag

Tag Documents Using NER

This utility allows you to tag one or a set of documents for named entities using Stanford's Named Entity Recognizer (NER).

The utility outputs .txt versions of the inputed documents with named entities wrapped around XML tags that show whether the named entity is a Location, a Person, an Organization, Money, a Percent, a Date, or a Time.
